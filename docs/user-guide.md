# TileTest User Guide

## Movement & Controls

- Click and drag to move your character
- Hold mouse button to move continuously
- Release to stop
- Movement indicator (yellow reticle) shows target
- Keyboard: I (inventory), Q (quests), H (help), Space (interact), Arrows (move), ESC (close dialog)

## Quest System

- Procedural quests based on technology domains
- Collect facts/items from NPCs/vendors
- Track progress, complete quests for points

## Inventory

- Max 8 slots
- Click PACK to view
- Drop items to free space
- Items used for quests or have point value

## NPCs & Vendors

- NPCs at booths, interact for facts/quests
- Each NPC tied to a technology domain

## Scoring & Progress

- Points for facts, quest completion, item collection
- Score shown top-left, persists across sessions

## UI & Accessibility

- Retro-styled, clear feedback
- Modal dialogs, paginated content
- Keyboard shortcuts, high contrast

## Debug Features

- Backtick (`) toggles debug mode
- Console logs for quest/domain events
- Asset caching disabled for dev

## Performance

- Optimized for browser play
- Efficient asset loading
- Minimal memory footprint
